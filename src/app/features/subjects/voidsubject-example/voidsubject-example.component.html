VoidSubject

Subjects Beginner

A VoidSubject (Subject with void type) is a specialized Subject that emits notifications without any associated value. It's perfect for signaling events where the occurrence matters, but no data needs to be transmitted.

  • Emits notifications without values
  • Perfect for event signaling (clicks, refreshes, etc.)
  • Lightweight - no data to store or transmit
  • Clear intent - signals "something happened" not "here's data"

Key Characteristics

🔔 Event Notification

Signals that an event occurred without transmitting any data.

⚡ Lightweight

No data to store, serialize, or transmit - just pure event signaling.

🎯 Clear Intent

Type signature makes it clear no value is expected or provided.

🔄 Multicast

All subscribers are notified simultaneously when event occurs.
