What is first?

The first operator emits only the first value (or the first value that meets a condition) emitted by the source Observable, then completes. It's perfect for scenarios where you only need the initial emission or the first match.

Key Concepts

🎯 Single Emission

Emits only one value then completes immediately

🔍 Optional Predicate

Can filter for first value matching a condition

⚠️ Error Handling

Throws error if no value matches (use default value to prevent)

⚡ Early Completion

Unsubscribes from source after first match

Common Use Cases

  • Initial Value: Get the first emission from a stream
  • First Match: Find first item meeting criteria
  • Quick Response: Get immediate result without waiting
  • Validation: Check if any value meets condition
