⏭️ skip Operator

🟢 Beginner

What is skip?

The skip operator ignores the first N values emitted by the source Observable. It's the opposite of take and is useful for skipping initial values or implementing pagination.

When to Use

  • ✅ Skip initial/default values
  • ✅ Pagination (skip to page N)
  • ✅ Ignore loading states
  • ✅ Skip warmup emissions

How It Works

1

Source Emits

The source Observable starts emitting values

2

Skip N Values

skip operator ignores the first N emissions

3

Emit Remaining

After skipping N values, all subsequent values are emitted

Real-World Use Cases

📄 Pagination

Skip to specific page of results

skip(page * pageSize)

🔄 Skip Initial State

Ignore default/initial values

skip(1)

⏳ Skip Loading

Skip loading state emissions

skip(1)

🎯 Warmup Period

Skip initial warmup emissions

skip(3)

✅ Best Practices

  • Combine with take for pagination (skip + take)
  • Use skip(1) to ignore initial form values
  • Great for implementing "load more" functionality
  • Consider skipWhile for conditional skipping
  • Observable continues after skipping N values
