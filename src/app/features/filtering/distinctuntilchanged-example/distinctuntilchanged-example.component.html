🔄 distinctUntilChanged Operator

🟢 Beginner

What is distinctUntilChanged?

The distinctUntilChanged operator only emits when the current value is different from the previous value. It's perfect for preventing duplicate consecutive emissions.

When to Use

  • ✅ Prevent duplicate API calls
  • ✅ Form value changes (skip if same value)
  • ✅ State management (avoid redundant updates)
  • ✅ Search optimization
